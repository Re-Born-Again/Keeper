# 📝 Keeper App

A simple note-taking app built with React and Material-UI, inspired by Google Keep.

## 🚀 Features

- Add and delete notes
- Expanding input area on focus
- Material-UI floating action buttons and icons
- Responsive design

## 🛠 Tech Stack

- React
- Material-UI
- HTML5/CSS3
